Introduction

In brief, this website is about endangered animals. Through this website we hope to spread awarness of animals that are endangered. Pulling data from various sources we have prepared graphs that will give people answers to what is going with these animals.

We strive to help people understand the severity of the sitation these animals face by being endangered. In addition we want to educate people on these animals, how they are endangered, and how you can possibly help save them!

These animals are critically endangered!!!

Pandas

There are only 1,864 pandas left in the world!

Black Rhinos

There are only 5,600 Black Rhinos left in the world!

Red Pandas

There are only less than 10,000 Red Pandas left in the world!
